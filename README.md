## Prerequisite

Install Hugo with [Homebrew](https://formulae.brew.sh/formula/hugo) üëá

```bash
brew install hugo
```

Install Yarn [v4.1.0](https://yarnpkg.com/getting-started/install)

```bash
corepack enable
yarn set version stable
yarn install
```

## Customization
